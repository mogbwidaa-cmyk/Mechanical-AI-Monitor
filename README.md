Executive Summary
As a Mechanical Engineer with extensive experience in maintenance and production management, I developed this platform to bridge the gap between traditional mechanical engineering and Industry 4.0. This project demonstrates the application of AI and Computer Vision to enhance machinery reliability and reduce operational downtime.

Key Features
Real-time Health Monitoring: A dynamic dashboard built with Python and Streamlit to track machinery vitals (Vibration & Temperature).

Predictive Analytics: Implemented Machine Learning algorithms to calculate the Machine Health Score and forecast potential failures.

Industrial Standards: Logic integrated based on ISO 10816 standards for mechanical vibration assessment.

Visual Fault Detection: A Computer Vision module using OpenCV for automated detection of surface cracks and corrosion in mechanical components.


Automated Alert System: An integrated notification system that triggers Email Alerts for critical maintenance needs, ensuring business continuity.
